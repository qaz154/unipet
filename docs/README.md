# UniPet Docs

A starting point for the UniPet documentation site.

Until a static-site generator is wired up (VitePress / Docusaurus), this directory acts as the canonical, browsable source of truth for the project. All pages are plain Markdown so they render correctly on GitHub and in any future site build.

## Table of Contents

- [Getting Started](./getting-started.md) — install, launch, connect agents
- [HTTP API](./http-api.md) — local REST + SSE control surface
- [MCP Tools](./mcp.md) — tools exposed via `@unipet/mcp`
- [Themes](./themes.md) — schema, marketplace, validation
- [Renderers](./renderers.md) — CSS-Pixel / SVG / Sprite / Live2D
- [CLI](./cli.md) — `unipet install`, `doctor`, `theme`, `react`, `say`
- [Adapters](./adapters.md) — Claude Code, Cursor, Codex, Perception …

## Status

This is a documentation **skeleton**. Each linked page contains the minimum content needed to be useful today and is structured to grow without breaking links.
